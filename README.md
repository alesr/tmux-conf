# tmux-conf

My tmux configuration

## Usage

```bash
mkdir -p ~/.config/tmux
git clone https://github.com/alesr/tmux-conf.git ~/.config/tmux

export TMUX_CONF=$HOME/.config/tmux/tmux.conf

```
